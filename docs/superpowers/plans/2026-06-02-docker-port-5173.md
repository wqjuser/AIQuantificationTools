# Docker Port 5173 Migration Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move the Docker web service from default port `8080` to the original app port `5173`, then stop the legacy Vite dev server so local usage goes through Docker.

**Architecture:** `compose.yaml` owns the external port default through `AIQT_WEB_PORT`. `tools/docker_smoke.py` owns the default smoke target. README documents the Docker-first endpoint and keeps `AIQT_WEB_PORT` as an override.

**Tech Stack:** Docker Compose, Python smoke helper, npm/Vitest deployment contract tests.

---

## Scope

- Change the Compose default web port from `8080` to `5173`.
- Change the Docker smoke helper default base URL from `http://127.0.0.1:8080` to `http://127.0.0.1:5173`.
- Update README Docker examples.
- Stop the existing local dev service on port `5173`.
- Start Docker Compose so `http://127.0.0.1:5173/` serves the deployed app.
