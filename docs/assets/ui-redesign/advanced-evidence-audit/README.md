# 行情中心概览与高级证据 UI 验收

本轮修复针对行情中心的两个视觉与交互问题：自选概览被等高布局拉成巨型空卡，以及“高级功能与证据”使用原生 `details` 伪装成全屏浮层。

后续复验又发现顶栏与页面各保留了一个行情输入框、缓存状态贴住表格边界、K 线信息条缺少留白，以及高级工作区重复渲染整块主图和行情摘要。本目录同时保留这轮结构收敛后的验收结果。

## 调整结果

- 自选概览改为紧凑的涨跌宽度摘要，并复用现有 watchlist 数据展示市场分布。
- 单个概览统计项高度由 `218.5px` 收敛为 `58px`，不再用空白撑高数字卡片。
- 高级入口并入顶栏操作组，不再悬浮在页面标题与刷新按钮之间。
- 高级工作区改用原生 `dialog.showModal()`，补齐标题、用途说明、关闭按钮、背景遮罩、焦点约束与内部滚动边界。
- 高级内容继续复用既有模型、API、store 和证据面板，没有引入第二套工作区实现。
- 删除页面内没有行为的第二个搜索框，只保留顶栏真实标的切换输入。
- 重新分配自选表末列宽度并居中缓存状态，桌面端左右留白均为约 `8px`。
- K 线底部信息条增加 `4px 10px 5px` 内边距，避免与图表和容器边界粘连。
- 高级工作区不再重复主图、行情卡与指标卡，只保留研究运营队列、数据源证据、扫描器和流程节点；桌面端收敛为右侧工作区，移动端维持单列滚动。
